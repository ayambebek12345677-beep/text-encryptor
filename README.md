# Text Encryptor

Encrypt and decrypt text using Caesar cipher, ROT13, and Base64 encoding.

## How to Use

**Step 1:** Open `index.html` in your browser.

**Step 2:** An onboarding tutorial appears on first visit. Click **Open Encryptor** (or Skip).

**Step 3:** Choose a method: **Caesar Cipher**, **ROT13**, or **Base64**.
- Caesar Cipher: shifts letters by a custom amount (default 3)
- ROT13: shifts letters by 13 (applying twice returns original)
- Base64: encodes text to ASCII characters

**Step 4:** For Caesar Cipher, set the **Shift** value (1-25).

**Step 5:** Type or paste text in the **Input** panel on the left.

**Step 6:** Click **Encrypt** to encode, or **Decrypt** to decode.

**Step 7:** The result appears on the right. Click **Copy** to save it.

**Step 8:** Click **Swap** to move output to input for quick decryption.

## Features
- Caesar Cipher with custom shift (1-25)
- ROT13 encoding
- Base64 encoding/decoding
- Encrypt and Decrypt both directions
- Swap input/output
- Copy to clipboard
- Educational info about each method
